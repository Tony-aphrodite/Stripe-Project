Prueba de seguridad

La prueba de seguridad es un servicio "echo" que se encargará de verificar que el mensaje y la firma correspondan a tu aplicación; asimismo retornará el mismo mensaje con nuestra respectiva firma, que también deberás verificar.

Objetivo

Generar un canal seguro para quien consume nuestras APIs, de tal manera que se implementa el firmado en cada una de las peticiones, y así mismo se entrega una firma para garantizar el no repudio.

¿En qué consiste?

Permite a las organizaciones:

  • Genera tu llave privada y tu certificado sigue estas instrucciones java o php.
  • Compártela con nosotros aquí.
  • Descarga nuestro certificado aquí.
  • Configura y utiliza cualquiera de nuestros clientes.

Laboratorio de Producción

Consume nuestras APIs en nuestro ambiente productivo una vez que tengas el acceso.

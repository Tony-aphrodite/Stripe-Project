You’ve found Truora's Collection! Dedicated to showing you all the ways that you can use Truora’s products from the API.

Truora’s API is organized around REST. Our API has predictable resource-oriented URLs, accepts form-encoded request bodies, returns JSON-encoded responses, and uses standard HTTP response codes, authentication, and verbs.

The following steps allow you to configure the collection appropriately for your use:

Prerequisites

  • Truora Account > API Key﻿

Let's start

Step 1:

Export the collection from Truora's public workspace within Postman:

Import the truora collection .json:

My Wordsáces > Import > Truora Collection.postman_collection.json

Step 2:

To execute requests, you'll need Truora account and supply your API Key, then set it as an environment variable within your workspace:

create a new environment > put variable name > Add your secret > save

then select the new environment you just created:

If your environment is set up correctly, you should see the value of your secret key if you hover over the secret_key variable in the Token field:

Everything is ready to start making a test call!!

You are now ready to make a test call. An easy first call is to create Enrollment for an email validation:

Add parameters to the call by clicking the Body tab, where you will see a list of available parameters. Select and fill in the required fields:

If your environment is set up you'll get a object back as the response to the call:
